# Rary

## Install

### Build

```bash
sudo make instal
sudo make clean
```

### Include as a subproject

```bash
git submodule add https://github.com/g0rbe/rary rary
```

## Include

Include the library with `-lrary`.

## Documentation

Documentation can be found in `docs/`.

## man pages

Install man pages:

```bash
sudo make install-man
```
